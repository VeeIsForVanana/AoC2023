THE PROBLEM:

    Input Constraints:

        Some file identified by filename containing some number of games formatted as such:
            "Game x: <round1>; <round2>; ... \n"

            For some number of rounds with comma-delimited counts of items formatted as such:
                "<count> <color>"

                For some color in the set:
                    "red", "green", "blue"

    Output Constraints:

        The sum of all x such that Game x is possible on a bag with 12 red, 13 green, and 14 blue items
